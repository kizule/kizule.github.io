CSS se koristi za definisanje stila web stranica, uključujući dizajn, izgled i varijacije prikaza web stranice.

Prema ovome, CSS se može koristiti kako bi prilagodili stranicu da se ispravno prikazuje na različitim uređajima, poput mobilnih telefona, računara i slično.

CSS je napravljen da bi se sve ovo navedeno olakšalo, jer se prvobitno pre verzije 4.0 HTML-a koristio sam HTML i za ovu svrhu.

Međutim, bilo je komplikovano da se za svaki element (pogotovo kada ima više HTML fajlova i sajt je veći) posebno navodi stil, te je napravljen CSS.

Koristeći CSS, u odvojenom fajlu možemo definisati kako će izgledati nešto na stranici, te samo u HTML fajlu navedemo odgovarajućom oznakom vezu ka odgovarajućem CSS fajlu.
